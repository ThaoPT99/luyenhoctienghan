# 🇰🇷 Korean Study Hub

Ứng dụng học tiếng Hàn toàn diện trên máy tính, tích hợp **DeepSeek AI** làm gia sư cá nhân.

## 🚀 Cách sử dụng

1. **Mở ứng dụng:** Double-click vào file `index.html` để mở trong trình duyệt.
2. **Cấu hình AI (tuỳ chọn):** Vào tab "AI Trợ giảng" → Nhập DeepSeek API Key để học với AI.
3. **Bắt đầu học:** Vào tab "Bài học" → Chọn Bài 1 để bắt đầu.

## 📖 Tính năng

| Tính năng | Mô tả |
|-----------|-------|
| 📖 **Bài học** | 10 bài học từ cơ bản đến sơ cấp, có từ vựng + ngữ pháp + ví dụ |
| 🃏 **Flashcard** | Lật thẻ ghi nhớ từ vựng với hệ thống ôn tập thông minh |
| ✍️ **Bài tập** | Trắc nghiệm từ vựng, ngữ pháp, dịch câu |
| 🤖 **AI Trợ giảng** | ChatGPT bằng DeepSeek AI — hỏi bất cứ điều gì về tiếng Hàn |
| 📚 **Từ vựng** | Danh sách từ vựng có tìm kiếm và phân loại |
| 📊 **Tiến độ** | Theo dõi streak, bài đã học, điểm số, lịch học |
| 💾 **Tự động lưu** | Tất cả tiến độ được lưu trên máy tính (localStorage) |

## 🔑 Lấy DeepSeek API Key (Miễn phí)

1. Truy cập [platform.deepseek.com](https://platform.deepseek.com/)
2. Đăng ký tài khoản
3. Vào API Keys → Tạo key mới
4. Copy key (dạng `sk-...`) và dán vào ứng dụng

> 💡 DeepSeek cung cấp $5 credit miễn phí khi đăng ký, đủ dùng rất lâu cho việc học!
